# FXNote Web Application

A fullstack app that lets you record your trades with foreign exchange and analyse your trades.

## URL
https://fxnote.net/

## Features
- Record trades via csv imports or by manually inserting trade details.
- Provides analysis tool to analyse your trades easily
- Subscription payments with Stripes payment portal

## Tech Stack
- UI: TypeScript
- Framework: Next.js
- Database: Supabase
- Payment system: Stripe
